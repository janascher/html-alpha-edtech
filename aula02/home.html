O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizado para definir a estrutura do seu conteúdo em um website, ou seja, ele não é uma linguagem de programação. Sem isso, o navegador não conseguiria mostrar textos como elementos ou carregar imagens e demais conteúdos.

O que são tags?

As tags são utilizadas para criar um elemento, como a fonte da letra, as quebras de linhas e etc. A maioria dos elementos HTML são compostos por uma tag de abertura (<) e uma de fechamento (/>). Mas também há aquelas de estrutura única que não possui a tag de fechamento.

Algumas tags HTML

  • <!-- -->: cria um comentário em qualquer parte do código;

  • <meta />: traz dados básicos sobre o documento, como autor e data da criação;

  • <html><html/>: está sempre no início e fim da página web. Ela limita onde o código começa e termina;

  • <head><head/>: usada para definir o início e fim d cabeçalho de um documento HTML;

  • <title><title/>: define o título do documento HTML. É o nome que aparece na aba da janela do navegador;

  • <body><body/>: é a parte principal da página web, ou seja, contém todo o conteúdo para mostrar ao público que visita a página;

  • <h1><h1/>: os elementos de cabeçalhos permitem definir o que são títulos ou subtítulos. HTML possui 6 níveis de título, entretanto, é mais comum utilizar de 3 a 4:

  • <p><p/>: define o início e fim de uma parágrafo;

  • <br />: define quebra de linha simples em um texto;

  • <a><a/>: cria um link e atribui elementos a ele, como href para vincular uma URL ao link;

  • <ol><ol/>: cria listas ordenadas, podendo ser numérica ou alfabética;

  • <ul><ul/>: cria listas não ordenadas (com marcadores);

  • <li><li/>: define um item de lista;

  • <img />: utilizada para incorporar uma imagem na página HTML, sendo que esta tag possui dois atributos obrigatórios: src (contém o caminho para o arquivo da imagem) e alt (especifica um texto descritivo aos usuários que não podem visualizar a imagem).

Como usar os cabeçalhos h1, h2, h3, h4, h5 e h6?

Os cabeçalhos podem ser trabalhadas em uma escala de 1 a 6, a fim de estabelecer uma organização hierárquica das informações, ou seja, a tag H1 é a mais importante da página e precisa ser atribuída ao título principal do conteúdo. Já as tags H2 são vinculadas aos subtítulos, as tags H3 a outros trechos de destaque, porém com menor relevância as tags H2, e assim por diante. Entretanto, raramente será utilizado as tags H4, H5 e H6.
